# 2.1.1 — 学生端旧版布局回迁

- 学生端 Vue 页面按旧静态前端重新迁移版式：首页、登录、冒险地图、随机练习、极限挑战、学习记录、成就、排行榜、PVP 大厅与 PVP 对战统一切回旧版卡片化学习界面语言。
- `HomeView` 恢复旧版结构：顶部品牌栏、三张主模式卡片、记录/成就/排行榜/竞技场次级入口，以及账号状态面板。
- `GameView` 重做为旧版学习流：左侧单元、右侧关卡、题卡式答题、练习解析区、极限挑战选择与结果页；并修正极限模式最后一题后无法推进下一挑战段的问题。
- `RecordsView`、`AchievementsView`、`LeaderboardView`、`PvPLobbyView`、`PvPBattleView` 全部补回更完整的版式层次，恢复总览指标、成就卡片、榜单行、竞技场概览和对战反馈。
- `AppShell` 升级为统一学生端顶部外壳：显示品牌、当前账号、首页/登录/退出操作。
- 排行榜前端改为支持匿名访问，游客也可直接查看公开榜单。
- 验证通过：`npm run build` 成功，`npm test` 3/3 通过。
