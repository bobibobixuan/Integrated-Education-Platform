# 2.1.4 — 学生端导航、账户中心与 PVP/刷题重构

- 学生端首页重构为真正的导航页：分成推荐入口、四个主模式块（开始冒险 / 随机练习 / 极限挑战 / PVP 竞技）、账户摘要块和次级入口（学习记录 / 成就墙 / 排行榜）。
- 新增独立个人中心页，首页账户卡只保留摘要；学生现在可以修改昵称、修改密码，并从个人中心统一退出登录。
- 后端新增 `PUT /api/auth/profile` 学生资料更新接口，前端认证 store 同步补齐昵称更新和改密后的会话刷新。
- 学习流从旧的 `/game` 巨型单页拆成多路由：冒险地图 / 作答 / 结果、随机练习入口 / 作答 / 总结、极限挑战入口 / 作答 / 结果。
- PVP 学生端拆成大厅 / 房间准备 / 对战 / 结算四段式页面；大厅加入房间列表和“我的当前房间”推荐区，房间页负责准备和成员状态，对战页收敛成题目主区 + 排名侧栏，结算页单独承接排名反馈。
- 新增学生侧只读房间列表接口 `GET /api/pvp/rooms`，用于大厅展示房间状态。
- 验证通过：前端 `npx vitest run --config vitest.config.ts` 5/5 通过，前端 `npm run build` 成功，后端 `backend/.uv-backend-venv` 环境下 `python -m pytest tests/integration/test_auth.py -q` 4/4 通过。
