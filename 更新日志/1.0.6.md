# 1.0.6

- 新增备用端口 8000 的打包配置，生成 `game_8000.exe`，无需管理员权限即可启动。
- `server/main.py` 重构：启动逻辑提取为 `run_server()` 函数，支持通过 `GAME_PORT` 环境变量自定义端口。
- 新增 `server/main_8000.py` 入口，默认端口 8000。
- 新增 `pyinstaller_8000.spec` 和 `build_8000.bat`，打包输出到 `dist/game_8000/`。
- 新增关卡制战力值系统（四维得分：通关/完美/速度/连击），取代现行+100累加制。
- 新增战力总榜和在线时长榜（原周活跃榜），支持 `?type=power` 和 `?type=weekly` 切换，包含 my_rank。
- 新增答题 Session 管理（start-session → next-question → answer），带时序校验和幂等保护。
- 新增防作弊系统：session 过期、答题时间下限（2s）、时序交叉验证、combo 重算、可疑 session 检测。
- 新增修改密码功能（PUT /api/auth/password），Token 版本化 + WebSocket 主动踢出。
- 新增学生账户管理（创建/编辑/禁用/恢复），支持 include_disabled 查询。
- 新增 WebSocket 鉴权缓存（10s TTL），减少 DB 查询压力。
- 新增 ScoreReviewDecision 表，支持教师裁决（approve/exclude/clear）。
- 新增 SessionQuestion / PlaySession 表，支持答题时序追踪。
- 新增 playing_session_id 字段到 AnswerRecord，实现 session 维度答题记录。
- 新增 power_score 字段到 UserStats，实现战力值存储。
- 新增 server/migrations.py 版本化迁移基础设施和战力值历史数据迁移。
- 新增服务器启动时自动执行迁移，无需手动运行脚本。
- 开启 SQLite WAL 模式，提升并发读写性能。
- 强制改密门禁：force_password_change 用户只能访问 /api/auth/me 和 /api/auth/password。
- 用户禁用后自动踢出 WebSocket 连接，防止已禁用用户继续在线。
- 前端新增战力值结算弹窗（四维得分拆解展示）、排行榜画面、修改密码弹窗。
- 教师后台新增学生 CRUD 操作按钮、添加学生按钮、启用/禁用状态筛选。
