# 2.0.7 更新日志

## 启动体验
- 启动服务器时自动打开浏览器到管理端页面（`/app/admin`），方便老师快速进入后台
- 启动时打印服务器横幅，显示首页地址、管理端地址和局域网地址

## 首页重建（Vue 3）
- 用 Vue 3 框架全新重建首页 `HomeView.vue`，视觉风格对齐旧版学生端首页
- 「Python 基础闯关」标题、三个行动卡片（开始冒险/随机练习/极限挑战）、底部导航（学习记录/成就/排行榜/竞技场）
- 登录状态感知：已登录显示用户名和进入学习/退出按钮，未登录显示登录/注册入口
- 路由调整：`/` → 新首页，`/login` → 登录页，登录页新增「返回首页」链接

## Excel 题库导入
- 管理端「题库导入」页新增 Excel 导入功能，支持 `.xlsx` / `.xlsm` 格式
- 智能表头识别：自动检测列布局，兼容「题目导入模板」和「题目列表」两种 Excel 格式
- 按「知识点」列自动分组：每个知识点自动创建一个单元，单元内所有题目放入一个关卡
- 支持的题型：单选题、多选题、判断题、填空题
- 一次导入最多 500 题，最多创建 20 个单元
- 后端新增 `POST /api/admin/import/excel` 端点，使用 openpyxl 解析 Excel 文件
- 单元名称留空时可从 Excel「目录」列自动提取

## 构建
- `pyproject.toml` 新增 `openpyxl` 依赖
- `pyinstaller.spec` 新增 `openpyxl` 隐藏导入
