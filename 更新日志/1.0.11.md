# 1.0.11

## 学生端 UI 重构

- 重构学生端整体视觉语言，统一首页、登录页、单元选择、关卡地图、排行榜、竞技场和学习档案页的界面风格。
- 学生端首页改为更明确的“学习作战台”布局，强化主线闯关、复盘和竞技场入口层级。
- 登录页补充学习档案与同步能力说明，入口信息更清楚。
- 单元选择页、地图节点、题目卡片、榜单面板和竞技场面板统一为同一套厚边卡片风格。

## 交互与适配

- 重做学生端卡片、按钮和榜单切换样式，提升信息分区和可点击反馈。
- 优化移动端布局，避免首页和记录页在窄屏下层级拥挤。
- 补充 `prefers-reduced-motion` 兼容，降低持续动画对低动态偏好用户的干扰。

## 缓存

- 学生端静态资源版本升级到 `1.0.11`，减少浏览器继续命中旧缓存导致页面样式不更新的问题。
